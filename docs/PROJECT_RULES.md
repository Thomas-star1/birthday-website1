# Regras do Projeto: Thomas Birthday Website

Este documento descreve as regras e diretrizes gerais para o desenvolvimento do website de aniversário do Thomas. O objetivo é criar uma experiência digital ultra-premium e futurista, inspirada em eventos de lançamento de produtos de alta tecnologia como Apple Event, Samsung Galaxy Unpacked e Tesla, com a qualidade visual de um site vencedor do Awwwards.

## 1. Visão Geral do Projeto

O website será uma celebração digital interativa para o aniversário do Thomas, com foco em uma estética moderna, animações fluidas e uma experiência de usuário imersiva. Para alcançar este objetivo, o projeto se baseará em um sistema de design rigoroso, detalhado no `DESIGN_SYSTEM.md`, e um guia de animações abrangente, presente no `ANIMATION_GUIDE.md`.

## 2. Tecnologias Principais

O desenvolvimento do website será realizado utilizando um conjunto de tecnologias modernas e performáticas. As principais ferramentas incluem **Next.js 15** para a estrutura da aplicação, **React** para a construção da interface de usuário, e **TypeScript** para garantir a robustez e manutenibilidade do código através de tipagem estática. Para estilização, será empregado o **Tailwind CSS**, permitindo um desenvolvimento ágil e responsivo. As animações serão orquestradas com **Framer Motion** e **GSAP (GreenSock Animation Platform)**, garantindo fluidez e alta performance. A experiência de rolagem será aprimorada com **Lenis**, e os ícones serão fornecidos pela biblioteca **Lucide React**.

## 3. Seções Essenciais do Website

O website deve apresentar diversas seções e funcionalidades cruciais para a experiência do usuário. Uma **Seção Hero Cinematográfica** servirá como introdução visualmente impactante, apresentando uma cidade futurista e um herói urbano mascarado original, inspirado em estética de aranha, sem recriar personagens protegidos por direitos autorais. Uma **Contagem Regressiva ao Vivo** para 1º de agosto manterá os visitantes engajados. Além disso, haverá uma **Lista de Presentes** interativa, organizada por **Categorias**. A experiência será enriquecida por **Animações Premium** em todo o site, o efeito de design **Glassmorphism**, um **Design Responsivo** completo para todas as telas e um **Modo Escuro** elegante como padrão.

## 4. Princípios de Design e Experiência do Usuário (UX)

Os princípios de design e UX são fundamentais para a criação de uma experiência memorável. A **Estética Futurista e Premium** deve evocar sofisticação, inovação e alta tecnologia. A **Imersão Visual** será alcançada através de gráficos de alta qualidade, tipografia moderna e um esquema de cores coeso. A **Performance Otimizada** é crucial, garantindo que o site seja rápido e responsivo, com animações suaves e carregamento eficiente. A **Acessibilidade** será uma prioridade, assegurando que o site seja utilizável por todos os usuários, seguindo as melhores práticas de WCAG. Por fim, a **Originalidade** é incentivada, buscando soluções de design e interação inovadoras que evitem clichês.

## 5. Diretrizes de Desenvolvimento

Para garantir a qualidade do código e a eficiência do desenvolvimento, algumas diretrizes são essenciais. A **Modularidade** será promovida através da organização do código em componentes reutilizáveis e bem definidos. A **Manutenibilidade** será assegurada por um código limpo, legível e bem documentado. A **Testabilidade** será priorizada, facilitando a verificação da funcionalidade do código. Por fim, a **Segurança** será implementada seguindo as melhores práticas de desenvolvimento web.

## 6. Colaboração

A colaboração eficaz é vital para o sucesso do projeto. A **Comunicação** deve ser clara e constante entre os membros da equipe. Todas as alterações de código passarão por um processo de **Revisão de Código**. O **Versionamento** será gerenciado utilizando Git, com um fluxo de trabalho de branches claro, como Git Flow ou GitHub Flow.

Este documento serve como a base para todas as decisões de design e desenvolvimento. Qualquer desvio significativo deve ser discutido e aprovado pela liderança do projeto.
